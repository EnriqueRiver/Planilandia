Apuntes para usar la consola con GitHub

Comandos de la consola
cd (change directory) "directorio nombre de la carpeta o .. para salir de la carpeta"
mkdir (crear una carpeta) "Nombre de la carpeta"


GITHUB Commands
git add "nombre del archivo"
git commit -m "nombre del commit"
git push 
git status (te muestra todos los archivos modificados y/o nuevos dentro y fuera del commit)
git branch (para crear ramas dentro de github para administrar versiones) git branch "nombre del branch" (crea una nueva rama)
git checkout (sirve para salirte de las ramas dentro de la consola)

git Init (para preparar una carpeta con git hub solo es necesario una vez)
git remote add origin (no se para que sirve pero es necesario si quiero hacer un push solo una vez)
